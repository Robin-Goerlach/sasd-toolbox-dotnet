# Roadmap

## V0.1.0 – Numerics Preview

Goal: create a small, working and documented first release.

Scope:

- Core guard helpers
- Numerics: statistics, interpolation, bisection
- Demo console application
- Unit tests
- README, changelog and release notes

## V0.2.0 – Numerics Expansion

Possible additions:

- standard deviation;
- min/max helpers;
- numerical integration;
- more root-finding methods;
- more examples.

## V0.3.0 – Data Toolbox Concept

Possible additions:

- simple data structures;
- sorting/searching examples;
- index/file concept discussion.

No database abstraction should be added until the design is clear.

## V0.4.0 – Editor/Text Toolbox Concept

Possible additions:

- text buffer experiments;
- search/replace helpers;
- line/column utilities.

## V0.5.0 – Graphics or Demo Toolbox Concept

Possible additions:

- simple geometry helpers;
- demo programs;
- educational examples.

## Later: C++ sibling project

A C++ version should probably become a separate repository:

```text
sasd-toolbox-cpp
```

The C#/.NET project should not be forced to share one repository with C++ too early.
