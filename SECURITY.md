# Security Policy

## Supported versions

This project is currently in early preview. No production support policy exists yet.

## Reporting a vulnerability

Please do not publish security issues publicly before SASD-GmbH had a chance to review them.

For now, report security concerns through the official SASD-GmbH contact channel:

https://www.sasd.de

## Security note

This repository currently does not contain cryptographic functionality.

Encryption-related APIs are intentionally excluded from v0.1.0 because such functionality must be designed and reviewed carefully before publication.
