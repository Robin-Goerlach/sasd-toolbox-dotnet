# Changelog

All notable changes to this project will be documented in this file.

The format follows the spirit of Keep a Changelog, without overcomplicating the first preview release.

## [0.1.0] - Unreleased

### Added

- Clean repository structure for `SASD Toolbox for .NET`.
- `Sasd.Toolbox.Core` class library.
- `Sasd.Toolbox.Numerics` class library.
- `Sasd.Toolbox.DemoConsole` sample application.
- xUnit test projects for Core and Numerics.
- XML documentation comments for public APIs.
- Initial documentation, roadmap and release notes.

### Included numerics functionality

- Mean
- Median
- Population variance
- Sample variance
- Linear interpolation
- Bisection root finding

### Removed from the old Building Blocks experiment

- Old .NET Framework 4.5.2 project structure.
- Empty placeholder classes.
- Unfinished database helper.
- Unfinished encryption helper.
- Old logging experiment.
